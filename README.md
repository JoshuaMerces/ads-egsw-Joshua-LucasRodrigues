# PT-BR

## Atividade da faculdade
Aplicar os conceitos de Gerenciamento de Configuração de Software (GCS) por meio do uso de um sistema de controle de versão (Git e GitHub), documentando e rastreando modificações realizadas em um pequeno projeto.

## Objetivo
A atividade deve ser realizada em duplas e consiste em criar um repositório Git, simular o controle de versões de um pequeno projeto e registrar o processo de configuração.

## O projeto
Um simples timer de pomodoro: A Técnica Pomodoro é um método de gerenciamento de tempo desenvolvido por Francesco Cirillo no final dos anos 1980. A técnica consiste na utilização de um cronômetro para dividir o trabalho em períodos de 25 minutos, separados por breves intervalos.

# EN-US

## College Activity
Apply the concepts of Software Configuration Management (SCM) through the use of a version control system (Git and GitHub), documenting and tracking changes made to a small project.

## Objective
The activity must be carried out in pairs and consists of creating a Git repository, simulating version control for a small project, and recording the configuration process.

## The Project
A simple Pomodoro timer: The Pomodoro Technique is a time management method developed by Francesco Cirillo in the late 1980s. The technique involves using a timer to divide work into 25-minute intervals, separated by short breaks.
